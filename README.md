
# Phone shopping website

## Run Locally

Clone the project

```bash
  git clone
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the app

```bash
  npm run start
```


## Running Tests

To run tests, run the following command

```bash
  npm run test
```

To run the Lint test, please run the following command

```bash
  npm run lint
```
